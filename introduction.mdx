Zeblok Computational provides a single, pre-configured cloud environment where data scientists, data engineers, and AI engineers can access datasets, GPU-powered AI/ML frameworks, languages, tools, and AI algorithms from the Intelligence Marketplace — all from one place.

At the core of the platform is the **Ai-WorkStation**, built on JupyterLab, which lets you write and run code, mix in equations and visualizations, and collaborate with your team without setting up any infrastructure. For high-performance workloads, the **Ai-HPC-WorkStation** adds Slurm-as-a-Service and parallel processing across four or more GPUs.

The entire platform runs inside **Ai-MicroCloud™** — a turnkey, cloud-native AI environment that spans from Core to Edge and covers the full ML lifecycle: development, testing, training, and production deployment.

## Key components

- **Ai-MicroCloud™** — the underlying cloud-native environment that ties all components together
- **Ai-WorkStation** — a single unified GPU environment with 1 GPU, 1 vCPU, and 64 GB of storage
- **Ai-HPC-WorkStation** — scales to 4+ GPUs with Slurm-as-a-Service for parallel workloads
- **Ai-API™ Engine** — deploys your trained ML models as production REST APIs
- **Ai-Data Lake** — high-performance storage for structured and unstructured data at SSD speeds
- **Ai-MicroCloud™ Manager** — admin utility for RBAC, plans, namespaces, and datacenter management
