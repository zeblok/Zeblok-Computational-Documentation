The Zeblok Python SDK (version 1.1.1) lets you integrate external systems with the Zeblok AI-MicroCloud platform via code. You can automate resource management, deploy Ai-Pipelines across multiple datacenters, and manage object storage — all from your own scripts or applications.

## Installation

Install the SDK using pip:

```bash
pip install zeblok-sdk
```

The SDK is also available under the alternate package name `zeblok-py`:

```bash
pip install zeblok-py
```

## What the SDK supports

- **Plans** — create and manage resource plans defining CPU and memory allocations
- **Orchestration add-on (Ray clusters)** — spawn and manage Ray clusters for HPC workloads, model training, and large-scale computation
- **Namespaces** — create and manage namespaces to isolate and organize resources within your organization
- **Microservices** — deploy, query, and manage microservices running on AI-MicroCloud
- **Ai-Pipelines** — deploy Ai-Pipelines in auto-deploy and non-auto-deploy modes, list all pipelines, and validate pipeline configurations
- **Datasets and object storage** — manage datasets and interact with S3 (AWS), Blob (Azure), and MinIO object storage environments

### File transfers

You can upload and download files of any size directly to and from object storage via the SDK, whether you are working from your local machine or from within an Ai-MicroCloud environment.

### Multi-datacenter Ai-Pipelines

The SDK lets you deploy simultaneous Ai-Pipelines across multiple enterprise datacenters and collate results (Ai-Runs) at a single endpoint of your choice.
